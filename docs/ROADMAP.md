# Roadmap

## v0 - Local sync proof

- Local web server.
- QR join.
- WebSocket clients.
- Manual audio unlock.
- Clock offset estimation.
- Scheduled Web Audio test pattern.

## v0.1 - Real audio asset

- Add local audio upload or bundled royalty-free sample.
- Preload audio on every client.
- Schedule sample playback.
- Add readiness checks before play.

## v0.2 - Better host dashboard

- Host-only controls.
- Rename devices.
- Show sync quality.
- Show session readiness and warnings for high-latency clients.
- Kick device controls.

## v0.3 - Calibration

- Test pulse.
- Manual calibration slider per phone.
- Save per-device offset in local storage.

## v1 - Acoustic intelligence

- Microphone calibration.
- Echo and phase analysis.
- Volume balancing.
- Position/movement detection.

## Later

- Android wrapper or PWA.
- Play Store only when the web MVP proves value.
- Legal music strategy.
